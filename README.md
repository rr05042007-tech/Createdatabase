Createdatabase Using c++
This repository includes the complete source code, sample data files, and a README with instructions to compile and run the project. Users can either build the program using g++ or directly run the provided executable. Overall, it serves as a practical introduction to database systems using C++.
This database is created using basic c++ concept that is file i/o 
This database create for add professor details. 
In this database you have six(6) options 

--- Professor Menu ---
1. Add Professor
2. Display Professors
3. Search Professor
4. Delete All Professors
5. Delete Specific Professor
6. Exit
Enter choice:

After run this program you will these six option and Enter a number between these options according to your wish 

If you want to exit this program you need to type number six in Enter choice option
